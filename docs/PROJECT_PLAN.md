# PromptWeaver 项目计划

## 核心功能

1. 简单文本输入生成高质量图像
2. 支持图片上传 + 图生图
3. 智能迭代（DeepSeek 重构提示词 + Qwen-VL 辅助理解）
4. 预设库管理（风格、场景、构图等）
5. 显示优化后的提示词（教育意义）

## MVP 范围

- LangGraph Agent 流程
- DeepSeek 提示词优化模块
- Qwen 生图 + 图生图
- 基础前端界面（Vue3）
- 核心预设库（30-50 个）

## 后续计划

- 多轮对话迭代
- 自动优化建议
- Docker 一键部署
- 更多模型支持

详细设计请参考后续文档。
